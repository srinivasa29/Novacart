# 🛒 Novacart: Where Tech Meets Elegance

Novacart isn't just another e-commerce project—it’s a carefully crafted digital shopping experience. Born from a desire to blend absolute technical precision with a high-fidelity aesthetic, this full MERN stack platform bridges the gap between complex backend engineering and a smooth, "glass-like" user interface.

---

## ✨ What Makes Novacart Special?

- **🔐 Security You Can Trust**: I built a robust authentication layer from scratch using **JWT** and **BCrypt**. It’s not just about logging in; it’s about ensuring every user’s data is armored with industry-standard encryption.
- **💎 High-Fidelity Design**: Inspired by modern minimalism, the UI features **glassmorphism**, subtle micro-animations, and a completely custom-coded CSS architecture. It feels alive, responsive, and tactile.
- **🚀 Engineered for Speed**: No clunky refreshes. Thanks to a centralized **Axios** configuration and Vite, the transition between products and categories feels like a native app.
- **🧠 Intuitive State Management**: Whether you're adding items to your bag or curating a wishlist, the state stays perfectly synced across sessions—making the shopping journey effortless.
- **🏗 Scalable Foundation**: The backend isn't just a basic API. It follows a clean **Controller-Route-Model** architecture, making it ready for real-world growth and features like admin dashboards and advanced order tracking.

## 🛠 Hand-Picked Tech Stack

### The Frontend (The "Art")
- **React 18**: The heart of the interactivity.
- **Lucide Icons**: For that premium, sharp visual feel.
- **Custom CSS**: No shortcuts with Tailwind here—every pixel was placed with purpose.

### The Backend (The "Engine")
- **Node.js & Express**: The fast, reliable bridge.
- **MongoDB & Mongoose**: Flexible, powerful data modeling.
- **JWT & BCrypt**: The gold standard for modern web security.

## 🚀 Experience It Yourself

### 1. The Easy Way (Seed Data)
Don't start with a blank screen. I've included a seeder script that populates the database with high-quality product images and descriptions instantly:
```bash
cd server
node seeder.js
```

### 2. Standard Installation

**Backend Setup**:
```bash
cd server
npm install
# Add your MONGO_URI to a .env file
npm run dev
```

**Frontend Setup**:
```bash
cd client
npm install
npm run dev
```

## 📂 Thoughtful Architecture

I believe clean code is as important as a clean UI.
- `server/models`: Where the data "lives".
- `server/middleware`: The gatekeepers of security.
- `client/context`: The "brain" of the frontend shared state.
